<h3>课程目标：</h3>
<p>一个成熟的网站架构并不是一开始设计就具备高可用、高伸缩、高性能等特性的，它是随着用户量和业务线不断增加，基础架构才逐渐健壮的。在发展初期，一般都是从0到1，不会一上来就整一些大而全的架构！</p>
<p>本课程将从理论到架构再到实践，一步步带你熟悉中大型网站架构部署的流程及技术点。</p>
